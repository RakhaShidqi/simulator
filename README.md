# 🚀 Tools Simulator

Tools Simulator adalah aplikasi berbasis Node.js yang berfungsi sebagai simulator untuk berbagai keperluan (GPS, tunnel, dll.). Dikembangkan oleh [RakhaShidqi](https://github.com/RakhaShidqi).

---

## 📋 Prasyarat

Sebelum memulai, pastikan Anda telah menginstal:

- [Node.js](https://nodejs.org/) (minimal versi 14.x)
- npm (bundel bersama Node.js)
- Git (opsional, untuk cloning repositori)

---

## 🔧 Instalasi

Ikuti langkah-langkah berikut untuk menjalankan aplikasi di lingkungan lokal:

### 1. Clone repositori
```bash
git clone https://github.com/RakhaShidqi/simulator.git
cd simulator
```

###  2. Instal dependensi
```bash
npm install
```
Perintah ini akan menginstal semua package yang diperlukan berdasarkan file package.json.

### Penggunaan
Menjalankan Server
```bash
node server.js
```

### Mengakses aplikasi
Setelah server berjalan, buka browser dan akses:
```bash
http://localhost:3000
```
